Funções Logarítmicas


O que é?

Um número real b (0 < b ≠ 1), chamamo de função logarítmica de base b a função f de R*+, onde R associa a cada x o número logbx.

f(x) = logbx

O domínio é D(f) = (0, +∞) = ℝ*+ e a imagem é Im(f) = ℝ.

Uma função logarítmica é inversamente proporcional a uma função exponencial, pois se por exemplo, tivermos uma função exponencial de base 3 que leva 2 a 9, em uma função logarítmica de base 3 acontece o inverso, leva 9 a 2.

Gráfico

Para construir um gráfico de funções logarítmicas você pode atrivuir valores a x para encontrar seus pares ordenados em y. Porém, é importante saber quais valores de x podem ser atribuídos na função, tendo em vista as condições de existência dos logaritmos. Por isso é interessante estabelecer o domínio da função antes de fazer a construção do seu gráfico. Do mesmo modo, é relevante verificar a curvatura para ver se ela é crescente ou decrescente e quais as interseções dela com os eixos coordenados.

Assim, podemos estabeler alguns passos a serem seguidos antes da construção do gráfico.:

  1. Analisar o crescimento / decrescimento;
  2. Determinar o domínio;
  3. Achar as interseções com os eixos;
  4. Encontrar alguns pontos do gráfico.
  • Crescimento e Decrescimento

    • Seja b a base da função f(x) = logbx:

    ✓ b > 1 → Crescente → f(x) = log2x , g(x) = log5(2x - 3)

    ✓ 0 < b < 1 → Decrescente → h(x)=log1/2x , p(x) = log0,3(x + 2)


  • Domínio

    • Como dito anteriormente, não são todos os valores que o logaritmando pode assumir, ou seja, não são todos os valores que a variável x pode assumir, para que isso seja possível, é preciso que tais valores respeitem a condição de existência dos logaritmos. Assim, antes de simplesmente ficarmos depositando números aleatórios na função torcendo para que um desses faça parte do domínio dessa função, podemos resolver uma simples inequação. Isso será de grande ajuda para a contrução do gráfico mais a frente.


  • Interseção com o Eixo x: Raízes

    • Sempre quando o gráfico de uma função fizer uma interseção com o eixo x, nós iremos ter raízes reais. E para encontrar essas raízes, devemos utilizar a equação f(x) = 0.


  • Interseção com o Eixo x: Raízes

    • Nem toda função logarítmica intersecta o eixo das ordenadas, isso se dá por conta das restrições que existem no domínio. Se o domínio for somente de valores maiores que zero, não irá cruzar o eixo y.


Depois de realizar todos esses passo é só fazer a construção do gráfico e pronto, você tem sua função resolvida.
